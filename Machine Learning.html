Software and data pipelines

Examples of machine learning software that I developed

muritz

A directed graph alignment based on node-specific motif-role profiles that I help develop during my PhD. At its core, muritz is a Python wrapper for a C/C++ network-alignment code.
Repository

pymfinder

A Python package with which to find network motifs in complex networks and to analyze a growing list of network-motif related metrics that I help develop during my PhD.
Repository

RESOLDRE

An R package for generating "correlation-informed" null models, which combine the classic concept of null models and tools from joint statistical modeling in community ecology. I developed this package during my PhD, and it combines R and C++ code.
Repository

CITE-seq data pipeline

An open-source, packaged, portable and reproducible pipeline for RNA and protein expression to curate a growing corpus of CITE-seq datasets as a standarized set of 'SingleCellExperiments'. This pipeline builds from two distinct software components, the R data processing pipeline and a Shiny app designed to extend the list of available datasets.
Repository
