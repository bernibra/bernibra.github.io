Data rich, insight poor

A workshop that I co-organized with Fernando Cagua on how to use data-science to empower management and conservation. In this worshop, we taught how to create electronic surveys and interactive real-time dashboards to ecologists. The workshop has been presented several times, including at the EcoTAS 2017 conference and at ETHZ.

Scientific publications

Throughout my scientific career, I developed visualizations that have been published as part of several scientific articles, all of which were generated programatically using different software libraries. Examples of these include: network visualizations, representations of the output of Bayesian models, and geographical representations.

This website

Using D3.js, a JavaScript library for producing dynamic visualizations, I developed this website. Notice that, as opposed to regular websites, this simply displays an svg object that the user can interact with. The code is publicly available. That said, this was developed when I was starting my Ph.D. and as a way to learn D3.js; therefore, the code is messy and poorly documented (pending a clean-up).

Single-cell data analysis

This is one of the many examples of my use of R Shiny or Python Dash to create interactive web applications. In this particular case, I developed tagtango, a powerful web application designed for the comprehensive analysis and comparison of multiple cell annotations or clusterings. Examples of the use of this application were published at Bioinformatics.
